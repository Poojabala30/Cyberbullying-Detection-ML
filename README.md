# Cyberbullying Detection using Machine Learning

This is a basic machine learning classification project.

## Project Description
The project detects whether a given text contains cyberbullying or not using a Logistic Regression model.

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Jupyter Notebook

## Dataset
Cyberbullying Tweets Dataset (Kaggle)

## Output
The model predicts whether the input message is:
- Cyberbullying
- Not Cyberbullying
